# Three.js: Simple postprocessing

Please note, since using a custom threejs pipeline module. Instead of XR8.Threejs.xrScene() you will reference window.xrScene, which is the same object as xrScene.

This sample project is based on [How to use post processing in the three.js documentation](https://threejs.org/docs/#manual/en/introduction/How-to-use-post-processing)
