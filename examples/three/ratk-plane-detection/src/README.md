# three.js: Plane Detection

This project demonstrates how leverage the Reality Accelerator Toolkit (RATK) to add planes to an 8th Wall experience in three.js.

On supported devices, the planes are provided by RATK using the WebXR Plane Detection API.

Learn more about RATK:
- https://meta-quest.github.io/reality-accelerator-toolkit/
- https://github.com/meta-quest/reality-accelerator-toolkit

Learn more about the Plane Detection API:
- https://github.com/immersive-web/real-world-geometry/blob/main/plane-detection-explainer.md
- https://immersive-web.github.io/real-world-geometry/plane-detection.html
