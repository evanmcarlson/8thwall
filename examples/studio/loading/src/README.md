# Studio: Loading

This project demonstrates how to add loading UI to a Studio experience.

### Components

#### delete-on-reality-ready

This component uses a state machine to listen for the [realityready event](https://www.8thwall.com/docs/api/studio/events/general/#realityready) and manage loading UI.
