# Studio: Face Effects

Get started with 8th Wall Face Effects! This Niantic Studio example attaches 3D glasses to the detected 
face and applies a tattoo to the generated face mesh.

![](https://static.8thwall.app/assets/multiface-fm8afic2g8.gif)


### Project Info

The following attachment points are available to choose from:

forehead,
rightEyebrowInner,
rightEyebrowMiddle,
rightEyebrowOuter,
leftEyebrowInner,
leftEyebrowMiddle,
leftEyebrowOuter,
leftEar,
rightEar,
leftCheek,
rightCheek,
noseBridge,
noseTip,
leftEye,
rightEye,
leftEyeOuterCorner,
rightEyeOuterCorner,
upperLip,
lowerLip,
mouth,
mouthRightCorner,
mouthLeftCorner,
chin

### Reference Assets

Download 'uv-black.png' or 'uv-bright.png' from ```/assets/Textures/``` for face mesh UV reference 
images.

Download the face mesh model for reference in your 3D modeling software
[here](https://cdn.8thwall.com/web/assets/face/facemesh.obj).
